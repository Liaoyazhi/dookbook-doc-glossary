# 动态编程语言

**动态编程语言**是一种在**运行时**而不是编译时完成操作的编程语言。 例如，在JavaScript中，可以在程序运行时更改变量的类型或向对象添加新的属性或方法。

这与所谓的*静态编程语言*相反，在静态编程语言中，通常无法进行此类更改。

!!! note
    请注意，尽管编程语言的动态/静态属性与动态/静态类型之间确实存在联系，但两者远非同义。
