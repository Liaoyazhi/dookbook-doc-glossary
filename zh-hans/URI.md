# 统一资源标识符 (Uniform Resource Identifier, URI)

**URI**（统一资源标识符）是一个指向资源的字符串。最通常用在[[URL]]上来指定Web上资源文件的具体位置。
相比之下，*URN*是在给定的命名空间用名字指向具体的资源，如：书本的*ISBN*。
