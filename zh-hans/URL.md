# Uniform Resource Locator (URL)

**统一资源定位器**（*URL*）是指定在Internet上可以找到资源的位置的文本字符串。

在HTTP的上下文中，**URL**被叫做”*网络地址*“或“*链接*”。你的浏览器在其地址栏显示URL，例如 `https://developer.mozilla.org`。

URL也可用于文件传输（**FTP**），电子邮件（**SMTP**）和其他应用。
